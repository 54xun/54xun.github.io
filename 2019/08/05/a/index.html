运算符

Published on 2019 Aug 05 21:24:40
Last Updated on 2019 Aug 05 22:42:51

运算符就是处理不同类型的数据

算数运算符

加法 +
减法/负号 -
乘号 *
% 计算出余数，值为整数
/ 计算出结果，值可能是小数，整数
. 用来并置字符串，就是拼接

赋值运算符

x=y –> x=y 左操作数被设置为右侧表达式的值
x+=y –> x=x+y 加
x-=y –> x=x-y 减
x=y –> x=xy 乘
x/=y –> x=x/y 除
x%=y –> x=x%y 模
x.=y –>x=x.y 连接两个字符串

比较运算符

因为本来比较运算就是用来数字进行比较的，所有说，遇到字符串和数字比较时候，就是把字符串转成数字，然后再比较
==和===比较特殊，可以用来对两个字符串进行比较，还能比较布尔值，空，数组 不会进行类型转换
例： ‘1232qqssdsq1314’==>123 ,’adwfds124223243’==>0
类型转换：字符串转数字

逻辑运算符

就是进行与或非运算
and或&& or或|| not或!
它们的结果都是bool值

递增/递减运算符

三元运算符

语法格式：(expr1) ? (expr2) : (expr3)

错误控制运算符

PHP中使用错误控制运算符‘@’来把表达式可能产生的任何错误信息都被忽略掉

字符串连接运算符

if语句

语法：
if (条件)
{
条件成立时要执行的代码;
}

if…else语句

if (条件)
{
条件成立时执行的代码;
}
else
{
条件不成立时执行的代码;
}

if…elseif…else语句

if (条件)
{
if 条件成立时执行的代码;
}
elseif (条件)
{
elseif 条件成立时执行的代码;
}
else
{
条件不成立时执行的代码;
}
