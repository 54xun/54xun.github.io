数据库

Published on 2019 Aug 12 22:04:22
Last Updated on 2019 Aug 13 16:01:15

在当今的互联网中，最常见的数据库模型主要是两种，即关系型数据库和非关系型数据库

关系型数据库：一行存放一个值
MySQL、sql server（运行的是一个数据库服务，可以放表、文件）/mssql、sqlite、oracle、access（单文件方式存储）、postgresql、db2
非关系型数据库：一个键存放一个值
MongoDB（非关系与关系结合的数据库）、Redis（缓存服务器，中转一般数据不是特别重要）
连接MySQL：
Cmd连接：C:\phpstudy_pro\Extensions\MySQL8.0.12\bin这个目录下输入cmd，在cmd中输入mysql -uroot -p 密码是root

MySQL命令

查看有几个数据库
show databases;

创建数据库
create database wordpress;

删除数据库
drop database wordpress;

选择数据库
use mysql;

查看数据库里的表
show tables;

创建表
create table test.tb_users(
id int(0) NOT NULL AUTO_INCREMENT,
uname varchar(30) NOT NULL,
upass varchar(50) NOT NULL,
img varchar(255) DEFAULT ‘/images/acativ.jpg’,
PRIMARY KEY (id)
);
插入数据
INSERT INTO mynews.tb_users(
username,password,nick_name
)VALUES(
1,2,3
);

删除数据
DELETE FROM mynews.tb_users WHERE id = 2
删除整个库里的数据
DELETE FROM mynews.tb_users;
or(两条都删掉)
DELETE FROM mynews.tb_users WHERE username = ‘ww’ OR id = ‘15’;
and（两条都要满足才能删掉)
DELETE FROM mynews.tb_users WHERE username = ‘ww’ AND id = ‘15’;

修改数据
UPDATE mynews.tb_news SET title = ‘bb’ WHERE id = 1;(修改所有的数据：where = 1; 修改多个字段用逗号隔开 改两个相同字段，执行的后一个)

查询数据
SELECT title,author FROM mynews.tb_news;
